# Federated-Learning-Demo
Demo lại một phần của paper Communication-Efficient Learning of Deep Networks from Decentralized Data
**Mô hình MLP và CNN :**

python main\_nn.py

**Học liên kết với MLP và CNN:**

python main\_fed.py
